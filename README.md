# Udacity Blockchain Capstone

The capstone project, building upon the knowledge gained in the course in order to build a decentralized housing product.

### Prerequisites / Software used

```
JSON package
Truffle v5.1.21
Solidity v0.5.16
Docker 2.2.0.5
Zokrates
Metamask 7.7.8
Infura
OpenSea
```

### Installation

1. Download the app
2. Login to infura and MetaMask
3. Customize infuraKey and Infura Project ID in truffle.js
4. npm init and npm install package.json
5. Truffe develop, compile, migrate --reset and test
6. Migrate to Rinkeby network
7. Mint tokens using myetherwallet.com
8. Import and trade tokens on https://rinkeby.opensea.io/
